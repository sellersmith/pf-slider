## Installation

`yarn add slider-x`

Documentation will update asap!

## Basic Usage

Import slider-x

```
import 'slider-x'
import 'slider-x/dist/slider-x.css'
```

### All options available (currently)
```
defaultOptions = {
  slidesToShow: 1,
  slidesToScroll: 1,
  gutter: 0,
  autoPlay: true,
  autoPlayDelay: 3000,
  duration: 450,
  loop: true,
  draggable: true,
  paginationStyle: 'pagination-style-1',
  navStyle: 'nav-style-1',
  adaptiveHeight: false,
  height: 400,
}
```

Enjoy :)
